# Słownictwo angielskie B1–B2 — 3000 haseł

## Zmiany w tej wersji

- Kategorie mają czytelne nazwy, np. `B1 — Rzeczowniki 1/9 (A–C)`.
- Hasła są pogrupowane według poziomu oraz części mowy.
- Pole `question` zawiera wyłącznie angielskie słowo lub wyrażenie.
- Usunięto zdanie `Co oznacza angielskie słowo lub wyrażenie…`.
- Wszystkie 3000 pytań zachowują cztery odpowiedzi, poprawny `correctAnswerId`
  i wyjaśnienie.

## Import

Do aplikacji importuj pliki z katalogu `categories`.

Nie importuj plików `manifest.json` ani `README.md`.
