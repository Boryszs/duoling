# Słownictwo angielskie B1–B2 — 3000 haseł

Zestaw jest gotowy do importu do aplikacji English Quiz.

## Zawartość

- 30 kategorii JSON,
- 100 pytań w każdej kategorii,
- łącznie 3000 unikalnych haseł,
- 1500 haseł B1 i 1500 haseł B2,
- cztery unikalne odpowiedzi w każdym pytaniu,
- poprawne `correctAnswerId`,
- objaśnienie z poziomem i częścią mowy.

Do aplikacji importuj wyłącznie pliki z katalogu `categories/`.
Nie importuj plików `manifest.json`, `README.md` ani `SOURCES_AND_LICENSES.md`.

Poziomy B1/B2 są edukacyjną klasyfikacją opartą na CEFR-J i danych CEFR; nie istnieje jedna oficjalna, zamknięta lista wszystkich słów CEFR.

Walidacja techniczna: 0 błędów; 3000 unikalnych angielskich haseł.
